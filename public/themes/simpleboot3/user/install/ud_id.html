什么是UDID？

UDID（Unique Device Identifier）中文意思是设备唯一标识，它是苹果iOS设备的唯一识别码，每一台iOS设备的UDID都是独一无二的。

如何获取UDID？

扫描此二维码后点击访问页面的获取UDID，将开始安装描述文件，期间可能会要求输入您的锁屏密码，用于验证是否是您本人安装，安装完成后，即可在新地址中，获取到您的UDID。
